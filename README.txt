## Description

The Student List Management System is a command-line program written in C that allows users to create, manage, and manipulate a list of student records. It provides basic functionalities such as adding students, deleting students by last name, and printing the list in both forward and backward directions.

## Features

- Create a new student record with last name, first name, student ID, classification, and graduation year.
- Append a student record to the end of the list.
- Delete student records by last name.
- Print the list of student records in both forward and backward directions.
